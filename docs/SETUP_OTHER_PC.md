# Setup da un altro PC

Per collegare il repository da un altro computer:

1. Clonare il repository
```bash
git clone https://github.com/AndreaBencini90/GraphStreamer.git
cd GraphStreamer
```

2. Accedere a GitHub
Usare le proprie credenziali GitHub oppure un token personale.

3. Sincronizzare il progetto
```bash
git pull origin main
git add .
git commit -m "Aggiornamento"
git push origin main
```
